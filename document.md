| 핵심은 최고 수익률 기준 패턴과 현재 수익률 기준 패턴을 분리해서 보는 거야. 둘이 의미가 다르거든. 표에 나온 종목들은 전반적으로 추세, 위치, 전략, 거래량, 수급 조합이 비슷하게 반복돼.  |  |  |  |  |  |
|  |  |  |  |  |  |
| 1) 최고 수익률 기준 구간별 공통 특성 |  |  |  |  |  |
|  |  |  |  |  |  |
| 구간 | 공통 추세/위치 | 거래량 특징 | 전략 특징 | 수급/점수 특징 | 해석 |
| 5~10% | 대부분 UP + Rising, 일부 Peak 섞임 | 0.5x~1.4x 수준이 많고, 폭발적 거래량은 적음 | 거의 전부 🚀 Momentum, 단기과열 자주 동반 | Antigrav/Decision/수급은 중간 수준 | 전형적인 중간급 모멘텀 종목군. “잘 오르긴 하지만 아직 압도적 강세주는 아님” |
| 11~20% | UP 비중 높음, Rising 중심이지만 Peak 비중도 증가 | 거래량 확인(✅) 비중 증가, 평균 거래량 배수도 더 큼 | 여전히 Momentum 중심, 과열 동반 잦음 | 수급과 승률이 5~10% 구간보다 조금 더 좋음 | 추세 + 거래량 가속이 붙는 구간. 급등은 잘 나오지만 추격 리스크도 커짐 |
| 21~30% | DOWN / Resting / Peak가 섞여 구조가 불안정 | 가장 큰 거래량 폭증이 자주 나타남 | 대부분 `Momentum | 단기과열` | 수급/승률/Decision이 기대보다 높지 않음 |

| 2) 현재 수익률 기준 구간별 공통 특성 |  |  |  |  |  |
|  |  |  |  |  |  |
| 구간 | 공통 추세/위치 | 거래량 특징 | 전략 특징 | 수급/점수 특징 | 해석 |
| 5~10% | 전반적으로 UP, Rising 많지만 Peak도 일부 존재 | 과도한 거래량보다는 보통 수준이 많음 | 대체로 Momentum, 과열은 일부만 남음 | 수급은 비교적 괜찮은 편 | “지금도 살아 있는 종목”이지만, 상승 유지형과 늦은 추격형이 혼재 |
| 11~20% | 거의 UP + Rising, Peak 거의 없음 | 1.0x~1.5x 정도의 적당한 거래량 | 대부분 Momentum, 과열 빈도 낮아짐 | 수급 50+가 많고 Antigrav도 무난 이상 | 가장 질 좋은 지속형 강세주 패턴. 네가 스캐너에서 진짜 노려야 하는 타입 |
| 21~30% | 오히려 DOWN / Peak / Resting 섞임 | 거래량은 매우 큼 | `Momentum | 단기과열` 비중 높음 | 수급은 낮거나 불균형 |

| 3) 급등 종목의 공통 특성 |  |  |  |  |  |
|  |  |  |  |  |  |
| 항목 | 공통 특성 | 의미 |  |  |  |
| 전략 | 거의 대부분 🚀 Momentum | 급등은 기본적으로 모멘텀형에서 많이 나옴 |  |  |  |
| 과열 여부 | ⚠️ 단기과열 동반 빈도가 매우 높음 | 급등주는 예쁘게 시작하는 것보다 과열과 같이 움직이는 경우가 많음 |  |  |  |
| 거래량 | 큰 급등일수록 거래량 폭증이 자주 나타남 | 거래량은 급등의 연료지만, 지속성까지 보장하진 않음 |  |  |  |
| 위치(Position) | Peak는 최고 수익률 구간에 자주 나타나지만, 현재 수익 유지형에선 불리 | 순간 급등엔 Peak가 많고, 오래 가는 급등엔 Rising이 많음 |  |  |  |
| 추세 | 현재 수익률이 잘 유지되는 종목은 대부분 UP | 진짜 좋은 급등주는 결국 UP 추세 유지 여부가 핵심 |  |  |  |
| 수급 | 50+가 유지형 종목에서 자주 보임 | 급등 자체보다 지속성에 더 중요 |  |  |  |
| 최종 조합 | UP + Rising + 과하지 않은 거래량 + 수급 50+ | 네가 스캐너에서 가장 우대해야 할 “지속형 급등주” 패턴 |  |  |  |

| 4) 실전적으로 바로 써먹는 해석표 |  |  |  |  |  |
|  |  |  |  |  |  |
| 분류 | 특징 | 스캐너 해석 |  |  |  |
| 반짝 급등주 | Momentum + 단기과열 + 거래량 폭증 + Peak/Resting + DOWN/NEUTRAL 전환 | 많이 튈 수는 있지만 재현성 낮고 추격 위험 큼 |  |  |  |
| 지속형 강세주 | UP + Rising + 거래량 1.0~1.5x + 수급 50+ + Peak 아님 | 계속 들고 갈 수 있는 좋은 급등주 패턴 |  |  |  |
| 늦은 추격형 | Peak + 단기과열 + 거래량 과도함 + 현재 수익은 높음 | 결과는 좋아 보여도 다음 진입용으론 부적합 |  |  |  |
| 건강한 초기 강세형 | UP + Rising + 거래량 적당 + 과열 약함 + 현재 수익률 11~20% 유지 | 가장 선호해야 하는 우량 급등 패턴 |  |  |  |
